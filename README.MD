# Shapedrawer

## Submodule Catch2 initialization

```
git submodule update --init --recursive --force
```

## Build application

```
make shapedrawer
```

## Build examples

```
make examples
```

## Build unittests

```
make unittest
```

## Build all

```
make all
```

## Debug with vscode

Make sure you have C++ tools plugin installed in vscode.

Please start vscode in the repo directory.

Build the project.

Start debugging.
